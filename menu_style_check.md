# 메뉴 언어 스타일 재점검 보고서

## 📋 현재 메뉴 구조 분석

### 네비게이션 메뉴 (7개)
```
1. 🅿️ 주차     (Parking)
2. 🗺️ 경로     (Route)
3. ✈️ 여행     (Travel)
4. 📅 플래너   (Planner)
5. 📢 민원     (Report/Complaint)
6. ℹ️ 안내     (Info)
7. 📋 예약     (Booking)
```

## 🔍 스타일 점검 결과

### ✅ 좋은 점
1. **이모지 일관성**: 모든 메뉴에 관련된 이모지 사용
2. **간결함**: 2-3자의 짧은 한글 단어
3. **명확함**: 기능을 직관적으로 표현

### ⚠️ 개선 필요 사항

#### 1. 언어별 스타일 불일치
- **한국어**: 명사형 (주차, 경로, 여행)
- **영어**: 일부 명사, 일부 동명사 혼재
- **일본어/중국어**: 체크 필요

#### 2. 용어 통일성
- "민원" → "Report" (영어에서 너무 formal)
- "안내" → "Info" (너무 축약됨)
- "플래너" → "Planner" (외래어 vs 영어)

#### 3. 이모지 적절성
- 📢 민원: 신고/불만 느낌 (🎯 또는 📝가 더 나을 수 있음)
- ℹ️ 안내: 너무 일반적 (🔍 또는 📍가 더 명확할 수 있음)

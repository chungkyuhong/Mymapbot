## 🎯 API 엔드포인트

### 챗봇 메시지 처리
```bash
POST /api/chatbot/message
{
  "message": "근처 맛집 찾아줘",
  "location": {"lat": 37.5665, "lng": 126.9780}
}
```

### 식당 예약
```bash
POST /api/chatbot/reservation
{
  "restaurantId": "R001",
  "date": "2026-02-05",
  "time": "18:00",
  "people": 4
}
```

### 메뉴 미리주문
```bash
POST /api/chatbot/preorder
{
  "restaurantId": "R001",
  "items": [
    {"name": "아메리카노", "quantity": 2, "price": 4500}
  ],
  "pickupTime": "30분 후"
}
```
